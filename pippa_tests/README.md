# Проверка расчетов Теории Pippa

Этот каталог содержит Python скрипты для верификации математических расчетов в Теории Pippa.

## Скрипты

### 1. `fractal_dimension_verification.py`
Проверка расчета фрактальной размерности D из энтропии Шеннона.

**Что проверяет:**
- Формула D = 1 + H/2, где H - энтропия трех квадрантов
- Значения вероятностей p1=0.905, p2=0.0475, p3=0.0475
- Вычисление энтропии H = -∑p_i log₂(p_i)

**Запуск:**
```bash
python fractal_dimension_verification.py
```

### 2. `particle_mass_verification.py`
Проверка расчетов масс элементарных частиц.

**Что проверяет:**
- Масса электрона как фундаментальный масштаб m₀ = 0.511 МэВ
- Расчет λ_μ = 1 / sqrt(m_μ² - m_e²) для мюона
- Массы протона и нейтрона
- Константу тонкой структуры α = 1/(R/λ_e)

**Запуск:**
```bash
python particle_mass_verification.py
```

### 3. `fine_structure_constant_verification.py`
Детальная проверка расчета константы тонкой структуры.

**Что проверяет:**
- Комптоновскую длину волны электрона
- Боровский радиус
- Формулу α = 1/(R/λ_e)
- Исправленную формулу с β коэффициентами

**Запуск:**
```bash
python fine_structure_constant_verification.py
```

### 4. `modes_equation_verification.py`
Проверка универсального уравнения мод (масса ↔ когерентность).

**Что проверяет:**
- Уравнение: P₁ C M + P₂ M₀ = M₀^(1-1/μ) M^(1/μ)
- Расчет когерентности для различных частиц
- Зависимость от фрактальной размерности μ

**Запуск:**
```bash
python modes_equation_verification.py
```

### 5. `run_all_verifications.py`
Главный скрипт для запуска всех проверок.

**Запуск:**
```bash
python run_all_verifications.py
```

- Нет ресурсоемких симуляций или больших массивов данных

- Рекомендации по уточнению параметров теории

## Структура теории

Теория Pippa имеет **11 базовых параметров**, из которых **6 независимых**:
- Фрактальная размерность D = 1.272
- Фундаментальный масштаб массы m₀ = 0.511 МэВ
- Параметры единого поля взаимодействия (D_U, κ_U, λ_U, g_U)
- Параметры состава поля (α, β, γ)

## Вывод параметров

Большинство параметров выводятся из первых принципов:
- D из энтропии информационных потоков
- m₀ как масса базовой стабильной моды
- α, β, γ из подгонки к константам связи
- Остальные параметры подбираются для описания масс частиц
